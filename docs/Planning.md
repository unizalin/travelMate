## 進行中的功能細節
1. **全球地址自動完成**：優化 `geocodingService.ts` 以提供更精準的地點搜尋。
2. **彈窗 UI 改善**：將所有 Modal 統一使用標準化的 `Transition` 與封裝好的 UI 組件。
3. **性能優化**：減少不必要的 Re-render，優化地圖 Marker 載入速度。

# Roundtable Council

## Goal
To simulate a multi-perspective discussion where you adopt three distinct expert personas to debate a solution before providing a final answer. This helps uncover hidden issues and find a balanced solution.

## Process

**Step 1: The Briefing**
- State the problem clearly.

**Step 2: The Discussion (Script)**
- Simulate a dialogue script where each member speaks 2-3 times.
- They should challenge each other's assumptions.
- *Example:*
    > **Alice**: We need a 3D globe here! It's so cool.
    > **Bob**: A 3D globe on mobile? The battery usage will be terrible.
    > **Charlie**: And we launch next week. Can we do a 2D map first?

**Step 3: The Consensus**
- The council must agree on a "Motion to Proceed".
- Summarize the final decision that balances all three views.

## Output Format
Please use blockquotes or a specific "Meeting Minutes" format for the discussion.
